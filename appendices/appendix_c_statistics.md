# Appendix C: Statistical Procedure

The quantitative analysis was conducted using a One-Way Repeated Measures ANOVA to track progress from V1 to V3.

### Analysis Details
- **Software:** Python (SciPy & Statsmodels)
- **Assumptions Checked:** 
    - Sphericity (Mauchly’s test)
    - Normality of residuals (Shapiro-Wilk test)
- **Post-hoc Analysis:** Bonferroni correction was applied for pairwise comparisons between drafts (V1-V2, V2-V3, V1-V3).

For the full analysis script and raw output, please refer to the `analysis/` folder in this repository.
